Gray-Scott Reaction-Diffusion SimulationThis project is a Python implementation of the Gray-Scott model, a mathematical framework used to simulate the formation of complex patterns in nature, such as coral structures, zebrafish stripes, and bacterial colonies.📌 Project OverviewAs a Mathematical Engineering student, I developed this simulation to demonstrate the application of Partial Differential Equations (PDEs) and numerical methods in Python. The simulation uses a discrete approximation of the Laplacian operator to model the diffusion and reaction of two chemical substances ($u$ and $v$).🧬 Mathematical BackgroundThe system is governed by two coupled non-linear partial differential equations:$$\frac{\partial u}{\partial t} = D_u \nabla^2 u - uv^2 + f(1-u)$$$$\frac{\partial v}{\partial t} = D_v \nabla^2 v + uv^2 - (f+k)v$$Where:$D_u, D_v$: Diffusion rates of substances $u$ and $v$.$f$: Feed rate (replenishment of $u$).$k$: Kill rate (removal of $v$).$\nabla^2$: The Laplacian operator, calculated using a five-point stencil.🛠 FeaturesNumerical Solving: Implementation of the Laplacian using np.roll for efficient grid-based computation.Dynamic Simulation: Real-time animation using Matplotlib's FuncAnimation.Presets: Includes specific parameters for simulating Coral, Zebrafish, and Bacteria patterns.💻 Engineering InsightThis project is more than a visual simulation; it represents the mathematical foundation of Convolutional Neural Networks (CNNs). The discrete Laplacian operator used here is a fundamental kernel in image processing and computer vision, showcasing how biological patterns and deep learning architectures share common mathematical roots.🚀 How to RunClone the repository.Install dependencies: pip install numpy pandas matplotlib.Execute the script: python Chemical-Reaction.py.
